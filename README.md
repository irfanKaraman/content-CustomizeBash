# NERD FONTS
> [Download NERD Fonts Manually/](https://www.nerdfonts.com/font-downloads)

- Linux Bash
  ```
  git clone --depth 1 https://github.com/ryanoasis/nerd-fonts
  cd nerd-fonts
  ```
  ```
  Install specific font:
  ./install.sh Hack
  ./install.sh JetBrainsMono
  ./install.sh Meslo
  or install all of them use:
  ./install.sh
  ```
